# `react-balanced`

Optically balance text in any HTML element.

## Installation

```zsh
yarn add react-balanced
```

## Usage

```tsx
import Balanced from 'react-balanced';

const MyComponent = () => (
  <Balanced as="h1">Lorem ipsum dolor sit amet, qui adipisicing elit.</Balanced>
);
```
